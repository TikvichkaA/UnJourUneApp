Politique de confidentialite / Privacy Policy

SecondMain ne collecte AUCUNE donnee personnelle.

Donnees collectees

Aucune. L'extension fonctionne entierement en local dans votre navigateur.

Fonctionnement

  • L'extension detecte le nom du produit sur la page que vous visitez
  • Elle effectue une recherche sur Le Bon Coin pour trouver des alternatives d'occasion
  • Les resultats sont affiches localement, rien n'est envoye a un serveur externe

Stockage local

L'extension utilise le stockage local de Chrome uniquement pour sauvegarder temporairement le dernier produit recherche. Ces donnees restent sur votre appareil et ne sont jamais transmises.

Permissions

  • activeTab : pour lire le nom du produit sur la page active
  • storage : pour sauvegarder temporairement les recherches
  • notifications : pour vous alerter si une bonne affaire est trouvee

Privacy Policy (English)

The extension only reads product names from web pages you visit to search for second-hand alternatives. No data is transmitted to any third party.
